HealthPocket is a web application that employers can use to help their employees choose the best health insurance plan offered by their company. With HealthPocket, users can:
- easily view health insurance options
- compare medication costs across plans
- receieve personalized insruance plan recommendations based off their medications

# To run this web app: 
### 1. Start the backend server
- cd into HethPocket/backend
- run 'python app.py'
### 2. Start the frontend application
once the server is running, open a seperate terminal
- cd into HealthPocket/frontend
- run 'npm install' if this is your first time running the app
- run 'npm start'

#### HealthPocket should open in your browser!

# How to use HealthPocket

- If you have an account, login with your username and password. 
- If you are new to HealthPocket, create an account (password must be at least 8 characters)
- upon login, you can add your personal medications under the 'edit medications tab'
- once you've added your medications, view the 'recommened insruance' tab to view your recommended plan
- view the 'medication price comparison' tab to view your medication coverage under your top three recommended plans
- view your profile to see an overview of your account
- click the red 'sign out' button on the top right corner of your dashboard to logout
- login and edit your medications at any time to view an updated recommendation!
